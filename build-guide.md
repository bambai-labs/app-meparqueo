# 🚀 Guía Rápida de Builds en EXPO (EAS)

## **iOS** 🍏

**Build de Producción**

```bash
eas build -p ios --profile production
```

**Subir a App Store**

```bash
eas submit -p ios
```

## **Android** 🤖

**Build de Producción (AAB)**

```bash
eas build -p android --profile production
```

**Generar APK (Preview)**

```bash
eas build -p android --profile preview
```
